### [leetcode 485. Max Consecutive Ones](https://leetcode.com/problems/max-consecutive-ones/)

Stan

```
    public int findMaxConsecutiveOnes(int[] nums) {
        int cur = 0;
        int max = 0;
        for(int i:nums){
            if(i==1){
                cur++;
            }else{
                max = Math.max(cur,max);
                cur = 0;
            }
        }
        max = Math.max(cur,max);
        return max;
    }
```
