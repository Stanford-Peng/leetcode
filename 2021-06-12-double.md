### [661. Image Smoother](https://leetcode.com/problems/image-smoother/)
Kang
```java
class Solution {
    public int[][] imageSmoother(int[][] img) {
        int rows = img.length-1;
        int cols = img[0].length-1;
        int[][] newimg = new int[rows+1][cols+1];
        
        //For each row
        for(int i = 0 ; i <= rows ; i++){
            //For each col
            for(int j = 0; j <= cols ; j++){
                int[] filter = new int[9];
                
                filter[0] = img[i][j];
                
                if(i > 0){ //Add top
                    filter[1] = img[i-1][j];
                    
                    if(j > 0){ //Add top-left
                        filter[2] = img[i-1][j-1];
                    }else {
                    filter[2] = -1;
                    }
                    
                    if(j < cols){//Add top-right
                        filter[3] = img[i-1][j+1];
                    } else {
                        filter[3] = -1;
                    }
                } else {
                    filter[1] = -1;
                    filter[2] = -1;
                    filter[3] = -1;
                }
                
                if(i < rows){ //Add bot
                    filter[4] = img[i+1][j];
                    if( j > 0){ //Add bot-left
                        filter[5] = img[i+1][j-1];
                    }else {
                        filter[5] = -1;
                    }
                    if(j < cols){ //Add bot-right
                        filter[6] = img[i+1][j+1];
                    } else {
                        filter[6] = -1;
                    }
                } else {
                    filter[4] = -1;
                    filter[5] = -1;
                    filter[6] = -1;
                }
                
                if(j > 0){ //Add left
                    filter[7] = img[i][j-1];
                } else {
                    filter[7] = -1;
                }
                if(j < cols){
                    filter[8] = img[i][j+1];
                } else {
                    filter[8] = -1;
                }
                    
                int sum = 0;
                int count = 0;
                //Loop nums for average
                for(int n = 0; n < 9 ; n++){
                    int num = filter[n];
                    if(num >= 0){
                        sum+= num;
                        count++;
                    }
                }
                int avg = sum/count;
                newimg[i][j] = avg;
            }
        }
        return newimg;
    }
}
```

### [Form Largest Integer With Digits That Add up to Target](https://leetcode.com/problems/form-largest-integer-with-digits-that-add-up-to-target/)
Guang  dp + 贪心
```java
class Solution {
    public String largestNumber(int[] cost, int target) {
      int[] store = new int[target + 1];
      Arrays.fill(store,Integer.MIN_VALUE);
      store[0] = 0; 
      for(int i = 1; i <= 9;i++){
          int temp = cost[i -1];
          for(int j = temp; j <= target;j++){
              store[j] = Math.max(store[j],store[j - temp] + 1);
          }
      }
      if(store[target] < 0)
       return  "0";
     String result ="";
      for(int i = 9,j = target; i > 0; i--){
          int temp = cost[i - 1];
          while(j >= temp && store[j] == store[j - temp] + 1){
            result += i;
            j -= temp;  
          }
      }
       return result;
    }
}
```
