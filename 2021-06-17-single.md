### [100. Same Tree](https://leetcode.com/problems/same-tree/)
Kang
```java
class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        
        if(p == null && q == null){
            return true;
        } else if(p == null && q != null){
            return false;
                
        } else if (q == null && p != null){
            return false;
        }
        
        if(p.val != q.val){
            return false;
        }
        
        if (isSameTree(p.left, q.left) && isSameTree(p.right, q.right)){
            return true;
        }
        return false;
    }
}
```
